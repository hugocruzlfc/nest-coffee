## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Notes

```bash
# Create a entity
$ nest g class coffees/entities/coffee.entity --no-spec

# Create a resolver
$ nest g resolver coffees

# Create a service
$ nest g s coffees
$ nest g resolver coffees/coffee-flavors --flat ,  👈🏻 this flag say to CLI to not create a folder and push the resolver to the coffees folder

```
